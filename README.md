# IPL Analytics Dashboard

Interactive IPL Analytics Dashboard built using Power BI analyzing IPL matches from 2008–2020.

## Project Overview

This project analyzes Indian Premier League (IPL) data from 2008–2020 using Power BI.
The dashboard provides insights into team performance, player statistics, venue performance, and match outcomes.

## Tools Used

• Power BI
• Data Cleaning
• Data Visualization
• Sports Data Analytics

## Key Insights

• Mumbai Indians have the highest number of IPL titles.
• Virat Kohli holds the record for the most runs scored (Orange Cap).
• SL Malinga leads with the highest wickets (Purple Cap).
• Teams choosing to field first after winning the toss win more matches.
• Venue analysis shows different match outcomes across stadiums.
